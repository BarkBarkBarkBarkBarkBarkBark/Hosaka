# 05.04 ¬∑ the reading tab

```
library ‚Äî reading material from the signal
```

The Reading panel is Hosaka's library. It's a tiny markdown reader
bundled with fourteen lore vignettes, all written in-character, all
part of the same slowly-unfolding history.

---

## opening the library

Two ways:

1. Click the **Reading** tab in the dock.
2. From the terminal:

```
/read                         # list the fragments
/read the-field-terminal      # open one in the reading panel
```

The terminal will say:

```
opening the-field-terminal in the reading panel.
```

‚Ä¶and the dock will switch to Reading.

## reading mechanics

Inside the panel:

- The list of fragments shows on the left (or top, on mobile).
- Click a title to open it.
- Markdown renders with `marked` ‚Äî headings, blockquotes, code, lists.
- Scroll to read; back to the list to switch.

From the terminal you can also:

```
/read order
```

‚Ä¶and get told:

```
the kindle relay isn't tuned yet ‚Äî coming soon.
```

(yes, that's a tease. yes, a kindle relay is coming. no, the orb won't
elaborate.)
